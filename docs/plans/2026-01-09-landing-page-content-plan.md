# Landing Page Content Specification

**Goal:** A comprehensive "Inspiration-First" landing page for a Text-Generation AI Prompt Library.
**Primary Focus:** Public Gallery (Discoverability) + Product Sales (New User Acquisition).
**Audience:** Prompt Engineers, Developers, Writers, Marketers.
**Constraint:** Text-Generation AI focus (LLMs like GPT, Claude, Llama) - **No Image Gen.**

---

## 1. Global Navigation Bar (Sticky Top)

*   **Visual Style:** Glassmorphism (blur backdrop), high contrast text.
*   **Left (Identity):**
    *   **Logo & Name:** "Prompt Library" (Icon + Text).
*   **Center (Navigation):**
    *   **Link 1:** "Discover" (Scrolls to Search/Categories).
    *   **Link 2:** "Features" (Scrolls to 'How it Works').
    *   **Link 3:** "Community" (External link to Discord/GitHub or internal forum).
*   **Right (Actions):**
    *   **Secondary Action:** "Log In" (Text link, prominent).
    *   **Primary CTA:** "Sign Up Free" (Button, Accent Color).

---

## 2. Hero Section: "The Inspiration Showcase"

*   **Objective:** Immediate intellectual hook. Show high-value text outputs to prove utility.
*   **Layout:** Split or Centered.
*   **Headline:** "Master the Art of Context." / "Build Better Prompts, Faster."
*   **Sub-headline:** "The definitive collection of optimized system prompts, chain-of-thought templates, and few-shot examples for GPT-4, Claude 3, and Llama."
*   **Key Dynamic Element: "The Text Carousel"**
    *   Instead of image previews, use **"Live Terminal / Chat Bubbles"**.
    *   **Animation:** Typewriter effect showing a complex prompt input → brilliant high-quality text output.
    *   **Carousel Items (Examples):**
        1.  *Coding:* "Refactor this React component to use Hooks..." → [Clean Code Output].
        2.  *Writing:* "Rewrite this email to sound professional yet firm..." → [Polished Text].
        3.  *Data:* "Extract JSON entities from this unstructured report..." → [Perfect JSON].
    *   **Metadata per Card:**
        *   **Model Badge:** "GPT-4" / "Claude 3.5 Sonnet".
        *   **Efficiency Score:** "Saved 20 mins".
        *   **Verified:** "✔ Verified by Community".

---

## 3. The "Action Core": Discovery Engine

*   **Position:** Immediately below Hero. The functional heart of the page.
*   **Component A: The Setup (Search)**
    *   **Input Field:** Large, centered, "Google-style" bar.
    *   **Placeholder Text:** "Search for 'Python Debugging', 'SEO Blog Post', 'Summarization'..."
    *   **Action Button:** Search icon + "Update Feed".

*   **Component B: Category Pills (Quick Filters)**
    *   **Header:** "Explore by Use Case:"
    *   **Tags/Buttons:** 
        *   `Software Development`
        *   `Creative Writing`
        *   `Business & Marketing`
        *   `Academic Research`
        *   `Roleplay & Persona`
        *   `Data Analysis`

*   **Component C: The "Prompt Card" Grid (The Result)**
    *   **Layout:** Masonry or Grid of cards.
    *   **Card Anatomy (Detail Density: Technical):**
        *   **Header:** Prompt Title (e.g., "Socratic Python Tutor").
        *   **Badges:** `System Prompt` | `Chain-of-Thought` | `Claude-Native`.
        *   **Preview:** 3-line snippet of the prompt logic:
            > *"Act as a senior python engineer. When the user provides code, do not give the answer immediately..."*
        *   **Stats (Social/Tech):**
            *   User Rating: ⭐ 4.9/5 (120).
            *   Tokens: ~450.
            *   Fork Count: 85.
        *   **Actions:** "Copy" (Quick), "View Details" (Full), "Star" (Save).

## 4. "Education" Section: How It Works & Why

*   **Header:** "Your Personal Prompt Engineering Workbench."
*   **Layout:** 3-Column Feature Blocks with Icons.
*   **Block 1: "Organize"**
    *   **Icon:** Folder/Tag.
    *   **Title:** "Stop Losing Your Best Prompts."
    *   **Text:** "Save, tag, and organize your prompt snippets in a searchable local-first vault. No more digging through chat history."
    
*   **Block 2: "Optimize"**
    *   **Icon:** Sparkles/Graph.
    *   **Title:** "Iterate to Perfection."
    *   **Text:** "Version control for your prompts. A/B test different phrasings and keep the one that delivers the best output."

*   **Block 3: "Sync"**
    *   **Icon:** Cloud/Sync.
    *   **Title:** "Everywhere You Work."
    *   **Text:** "Offline-first architecture means instant access on your desktop, syncing securely to the cloud when you're back online."
